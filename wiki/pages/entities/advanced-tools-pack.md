# Advanced Tools Pack

## Nature
Type : **pack prévu** — 8 systèmes avancés

## Description
Pack regroupant 8 systèmes utilitaires avancés :

1. **TEMPVOC** — salons vocaux temporaires
2. **FORMULAIRE** — formulaires customisables
3. **SUGGESTIONS** — système de suggestions avec vote
4. **AUTOBACKUP** — backup automatique de configs serveur
5. **AUTOREACT** — réactions automatiques sur messages
6. **TWITCH alerts** — notifications live Twitch
7. **REMINDERS** — rappels programmés
8. **CUSTOM TRANSFER** — transfert de données entre serveurs
