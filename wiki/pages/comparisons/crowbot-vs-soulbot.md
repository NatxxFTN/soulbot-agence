# CrowBot vs Soulbot

## TL;DR
CrowBot est une référence francophone populaire du segment bots 
Discord premium. Soulbot se positionne en challenger avec 
identité visuelle signature + Components V2 + Pack Forteresse exclusif.

## Axes de comparaison

| Critère | CrowBot | Soulbot |
|---|---|---|
| **Langue** | Français | Français |
| **Framework UI** | Embeds classiques | Components V2 exclusivement |
| **Identité visuelle** | Multi-couleurs | Rouge/noir signature |
| **Modèle commercial** | Premium payant | [À VALIDER NATHAN] |
| **Pack sécurité** | Standards (anti-raid, anti-link) | [[pack-forteresse]] (12 protections) |
| **Système permissions** | Rôles Discord standards | [[permissions-3-niveaux]] BotOwner/Buyer/Owner |
| **Commandes** | Très nombreux | ~132 base + packs prévus |

## Différenciation Soulbot
- **Design signature** (rouge/noir + Components V2)
- **Pack Forteresse exclusif** (12 détecteurs avec panel CV2)
- **Permissions 3 niveaux** (modèle premium)
- **Identité francophone premium** (ton punchy, emojis custom)

## Risques concurrentiels
- CrowBot a la notoriété et l'effet réseau
- Positionnement "premium français" partiellement occupé

## Conclusions
Soulbot ne doit pas essayer de battre CrowBot sur le volume de 
commandes. Il doit gagner sur la **qualité perçue** et l'**identité**.
