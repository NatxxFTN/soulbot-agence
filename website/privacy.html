Politique de Confidentialité

Comment nous collectons, utilisons et protégeons vos données

Dernière mise à jour : 7 juin 2026

🔒 Soulbot prend la protection de vos données au sérieux. Cette politique explique quelles données sont collectées, pourquoi, et comment vous pouvez exercer vos droits.

1. Responsable du traitement

Le responsable du traitement des données est l'exploitant de Soulbot (ci-après "le développeur"). Pour toute question relative à vos données, contactez-nous via le serveur Discord de support ou les canaux prévus à cet effet.

2. Données collectées

Soulbot collecte et stocke les catégories de données suivantes :

Catégorie Données Finalité
Identifiants Discord ID utilisateur, ID du serveur, ID des salons, ID des rôles Fonctionnement du bot, configuration, logs de modération
Messages Contenu des messages (uniquement dans le cadre des logs et des systèmes de modération/anti-spam/anti-insultes) Détection des infractions, logs d'activité, modération automatique
Actions de modération Avertissements, bannissements, expulsions, mute, suppression de messages Traçage des actions de modération, historique
Données de progression XP, niveaux, temps en vocal Système d'engagement et de classement
Données de participation Participations aux giveaways, sondages, suggestions, confessions anonymes Animation communautaire
Configuration Préférences du serveur (prefixe, salons de logs, rôles auto, protections activées) Personnalisation du bot par serveur
Données de diagnostic Logs d'erreurs, temps de réponse, utilisation des commandes Débogage, amélioration des performances

Ce que nous NE collectons PAS

  • ❌ Adresses email (sauf si fournies via OAuth2 Discord pour le tableau de bord)
  • ❌ Messages privés (sauf ceux envoyés volontairement au bot via commande)
  • ❌ Informations bancaires ou de paiement
  • ❌ Adresses IP
  • ❌ Données de localisation
  • ❌ Contenu des fichiers audio, vidéo ou images

3. Base légale du traitement

Le traitement des données est basé sur :

  • L'intérêt légitime : Fournir les fonctionnalités demandées par les utilisateurs (modération, logs, automatisations)
  • Le consentement : En invitant Soulbot sur un serveur et en utilisant ses commandes, vous consentez au traitement décrit dans cette politique
  • L'exécution du contrat : Les données sont nécessaires au fonctionnement du service

4. Stockage et sécurité

Les données sont stockées de la manière suivante :

  • Base de données locale (SQLite) sur le serveur hébergeant le bot
  • Fichiers JSON pour certaines configurations
  • Aucun stockage cloud externe : les données ne sont pas transmises à des tiers

Des mesures de sécurité sont mises en place pour protéger les données :

  • 🔐 Accès restreint aux fichiers de base de données
  • 🔄 Sauvegardes régulières
  • 🔑 Authentification requise pour le tableau de bord

5. Conservation des données

Les données sont conservées pour les durées suivantes :

Type de données Durée de conservation
Logs de modération (warns, bans, kicks) Durée de vie du serveur + 30 jours après retrait du bot
Logs d'activité (messages, commandes) 7 jours (rotation automatique)
Données de progression (XP, niveaux) Durée de vie du serveur
Configuration du serveur Durée de vie du serveur
Données de giveaways/sondages Jusqu'à la fin de l'événement + 30 jours

Lorsque Soulbot est retiré d'un serveur, toutes les données associées à ce serveur sont supprimées dans un délai de 30 jours.

6. Droits des utilisateurs (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

Droit Description
Droit d'accès Vous pouvez demander quelles données sont stockées à votre sujet
Droit de rectification Vous pouvez demander la correction de données inexactes
Droit à l'effacement Vous pouvez demander la suppression de vos données
Droit à la limitation Vous pouvez demander la restriction du traitement
Droit à la portabilité Vous pouvez recevoir vos données dans un format structuré
Droit d'opposition Vous pouvez vous opposer au traitement de vos données

Pour exercer ces droits, contactez-nous via le serveur Discord de support. Nous répondrons à votre demande dans un délai de 30 jours.

7. Partage des données avec des tiers

Soulbot ne vend pas, ne loue pas et ne partage pas vos données personnelles avec des tiers.

Les seuls cas de partage sont :

  • Discord Inc. : via l'API Discord, dont les conditions s'appliquent
  • Obligation légale : si requis par la loi ou une autorité compétente

8. Services tiers intégrés

Soulbot peut faire appel à des services tiers pour certaines fonctionnalités :

  • APIFlash : pour les captures d'écran de sites web (commande ;screenshot)
  • TinyURL : pour le raccourcissement d'URL (commande ;shorten)
  • OpenWeatherMap : pour la météo (commande ;weather)
  • Google Translate API : pour les traductions (commande ;translate)

Ces services ont leurs propres politiques de confidentialité. Les données qui leur sont transmises se limitent au strict nécessaire à la fourniture du service (ex: URL à raccourcir, ville pour la météo).

9. Sécurité des données

Nous mettons en œuvre des mesures techniques et organisationnelles appropriées pour garantir un niveau de sécurité adapté au risque, notamment :

  • 🔒 Stockage local des données (pas de cloud externe)
  • 🛡️ Restrictions d'accès à la base de données
  • 🔄 Sauvegardes automatiques régulières
  • 📋 Logs d'accès et de modifications
  • 🔑 Authentification forte pour le tableau de bord

10. Mineurs

Soulbot est destiné aux utilisateurs âgés d'au moins 13 ans, conformément aux Conditions d'Utilisation de Discord. Nous ne collectons pas sciemment de données personnelles de mineurs de moins de 13 ans. Si vous avez connaissance d'une telle collecte, veuillez nous contacter.

11. Modifications de cette politique

Cette politique de confidentialité peut être mise à jour périodiquement. La date de "Dernière mise à jour" en haut de cette page indique la date de la dernière révision. Les modifications prennent effet dès leur publication.

12. Contact et réclamation

Pour toute question concernant cette politique de confidentialité ou pour exercer vos droits, vous pouvez :

  • Nous contacter via le serveur Discord de support
  • Envoyer un message direct au développeur

Vous avez également le droit d'introduire une réclamation auprès de la CNIL (Commission Nationale de l'Informatique et des Libertés), l'autorité de contrôle française en matière de protection des données.
