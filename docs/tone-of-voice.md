# 🎙️ Charte de ton Soulbot

> Référence officielle maintenue par CV4 — Tone of Voice Guardian.
> Toute production textuelle (embeds, docs, changelog, support, directories)
> doit respecter cette charte. En cas de doute : le Tone Guardian tranche.

---

## On est

- **Clair** — Pas de jargon inutile. Un admin doit comprendre sans Google.
- **Chaleureux mais pro** — Tutoiement en général. Vouvoiement pour les messages admin sensibles (sanctions, permissions, facturation).
- **Efficace** — Zéro mot superflu. Si une phrase peut être coupée de moitié sans perdre le sens, on coupe.
- **Positif** — Même dans l'erreur, on propose une sortie : "essayons plutôt ça", "vérifie ici".

## On n'est pas

- **Familier** — "yo mec", "ptdr", "lol", "oklm" sont interdits.
- **Corporate** — "Nous sommes ravis de vous annoncer", "dans le cadre de", "à cet effet".
- **Alarmiste** — "ATTENTION !!!", "GRAVE ERREUR", "CRITIQUE". On pose, on guide.
- **Bavard** — Pas de disclaimers légaux dans un message bot. Pas de paragraphe pour dire bonjour.

---

## Règles absolues

| Règle | Interdit | Autorisé |
|---|---|---|
| Emojis | > 2 par message | ≤ 2, sémantiques |
| SMS | "stp", "mdr", "svp", "bcp" | Mots entiers |
| Erreurs | "Erreur : undefined" | "Le token n'est pas valide. Vérifie dans Paramètres > Sécurité." |
| Majuscules | TOUT EN MAJ | Première lettre uniquement |
| Après une erreur | Laisser l'utilisateur seul | Toujours proposer une action |

---

## Contextes et tons adaptés

| Contexte | Ton |
|---|---|
| Succès simple | Léger, direct. "✓ Rôle attribué." |
| Erreur utilisateur | Posé, action proposée. "Ce salon n'existe pas. Vérifie l'ID." |
| Action irréversible | Neutre, factuel. "Cette action est permanente. Confirme pour continuer." |
| Onboarding | Chaleureux, motivant. "Bienvenue ! Commence par configurer tes logs." |
| Premium upsell | Honnête, non intrusif. "Cette option est disponible en Premium." |
| Changelog public | Engageant, humain. Jamais "fix typo", toujours "Orthographe corrigée dans /tickets". |

## Vocabulaire préféré

| Terme à éviter | Terme préféré |
|---|---|
| Erreur survenue | Quelque chose n'a pas fonctionné |
| Utilisateur | Membre (contexte Discord) / Admin (si admin) |
| Channel | Salon |
| Server | Serveur |
| Problème | Souci (plus doux), ou reformuler |
| Veuillez | Pas de "veuillez" — aller direct |
| Effectuer | Faire |
