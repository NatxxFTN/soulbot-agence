# docs/ — Documentation Soulbot

## Relation avec wiki/

Le dossier docs/ (cette documentation) et le dossier wiki/ ont des 
rôles complémentaires :

- **docs/** : documentation stable du projet (ARCHITECTURE, ROADMAP, 
  DECISIONS, CHANGELOG, BRAIN). Édité à la main par Nathan 
  principalement.

- **wiki/** : base de connaissance vivante, maintenue par Claude Code. 
  Sources ingérées, synthèses, cross-références automatiques.

Les deux sont compatibles Obsidian et se référencent mutuellement.
